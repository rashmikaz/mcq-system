# 📝 Online MCQ System

This is a full-stack web application for conducting and managing multiple-choice exams (MCQs) online. It supports user registration, exam creation, question submission, and result evaluation.

---

## 🔧 Tech Stack

### 🖥️ Frontend
- React
- Redux Toolkit
- Axios
- Tailwind CSS
- Vite

### 🌐 Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL
- TypeScript
